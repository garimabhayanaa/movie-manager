# Movie Manager  

## Overview  
**Movie Manager** is a handy tool designed to help users **organize and track** their must-watch movies and TV shows. It provides an easy way to add, edit, and remove entries, ensuring you never miss out on your favorite entertainment. By leveraging a **local database**, Movie Manager streamlines watchlist management, helping users stay up to date with the latest releases.  

## Features  
- **Add, Edit, and Remove Entries:** Effortlessly manage your movie and TV show watchlist.  
- **Persistent Local Database:** Store your watchlist securely for future reference.  
- **Streamlined Watchlist Management:** Keep track of movies and shows from various OTT platforms in one place.  
- **User-Friendly Interface:** Intuitive design for seamless interaction.  

## Technologies Used  
- **Backend:** Node.js, Express.js  
- **Frontend:** EJS (Embedded JavaScript)  
- **Database:** Local database (PostgreSQL)  
- **Additional Tools:** CSS, JavaScript  

## Installation  

### Prerequisites  
- **Node.js** and **npm** installed on your system.  

### Steps  
1. **Clone the Repository:**  
   ```bash
   git clone https://github.com/garimabhayanaa/movie-manager
   cd Movie-Manager
2. **Install Dependencies:**
   ```bash
   npm install
3. **Run the Application:**
   ```bash
    node app.js
4. **Access the Web App:**
    Open your browser and visit http://localhost:3000 to use the Movie Manager.

## Usage
- **Add a Movie/TV Show**: Enter details and add it to your watchlist.
- **Edit Entries**: Modify information as needed.
- **Remove Movies/Shows**: Delete entries once you've watched them.
- **Stay Updated**: Keep track of new releases and upcoming content.
